Firebase Setup

Connect your admin panel to Firebase

⚠️
You need to configure Firebase to save data permanently. Follow the steps below. Until then, data is stored temporarily in your browser only.

Step 1 — Create a Firebase Project

  1. Go to console.firebase.google.com
  2. Click Add Project → name it mic-of-mithila
  3. Disable Google Analytics (optional) → click Create Project
  4. Click Firestore Database → Create database → Start in test mode
  5. Go to Project Settings → Add Web App → Copy the config object

Step 2 — Paste Your Firebase Config

Paste the config from Firebase console below. This is saved only in your browser.

Step 3 — Set Up Firebase Auth

  1. In Firebase Console → Authentication → Get Started
  2. Enable Email/Password sign-in method
  3. Go to Users tab → Add user → use your admin email & password
  4. Now use those credentials to log into this admin panel

Step 4 — Add Firebase SDK to admin.html

Once you have your config, replace the FIREBASE_CONFIG placeholder at the bottom of admin.html with your actual config, and uncomment the Firebase SDK imports. A developer can do this in 5 minutes.

📧 Need help? Share this file with a developer and show them this page — it explains everything they need.
