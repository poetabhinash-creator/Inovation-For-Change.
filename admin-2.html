Firebase Setup

Connect admin panel to your live website

📋 Step-by-Step Instructions

  1. Go to console.firebase.google.com → Create project → name it innovation-for-change
  2. Left sidebar → Build → Realtime Database → Create Database → Start in test mode → Choose asia-south1
  3. Go to ⚙️ Project Settings → Your Apps → Click </> Web → Register app
  4. Copy the firebaseConfig object and paste it below
  5. Click Save & Connect — your admin panel will now save to Firebase live!
  6. Upload index.html to your GitHub repo — it will automatically read from Firebase
